#### Installation

```bash
$ npm install
```

#### Before Start

###### 1. Setup ProstgresQL

```bash
sh script/setup.sh
```

> Customize the DB name by modifying the script. (initial value: 'project')

#### Running the app

```bash
# For development
$ npm run start:dev

# For production
$ npm run start:prod
```
